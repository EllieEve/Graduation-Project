Projeyi İndir

Teachable Machine ile Kullanım Rehberi

  1. İlk olarak yukarıdaki bağlantıya tıklayarak sınıflandırma projesini indirin.
  2. Teachable Machine sitesine girin.
  3. “Image Project” (Görsel Projesi) → “Standard Image Model” seçeneğini tıklayın.
  4. Sayfanın sağ üst köşesinden “Upload Project” (Proje Yükle) butonuna tıklayın.
  5. İndirdiğiniz `.tm` dosyasını seçerek projeyi yükleyin.
  6. Yükledikten sonra modeli düzenleyebilir, test edebilir ve eğitebilirsiniz.
  7. Son olarak, modeli “Export Model” ile dışa aktarabilir ve kendi projenize entegre edebilirsiniz.
